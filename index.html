LAMP Stack Catalog

I used a LAMP (Linux, Apache, MySQL, PHP) stack to build my current employer at the time, Michigan State University College of Osteopathic Medicine, a modern cataloging system for their use within their network of Learning Resource Centers. Using Koha ILS, I built a Drupal-based site which sat on top of a Koha catalog. The site is still in use today.

  • Client: Michigan State University College of Osteopathic Medicine
  • Date: May 2011
  • Service: Web Development, Cataloging, System Administration

Book Shelf Viewer

Using a book shelf display interface developed by Harvard Library Innovation Labs, I built a connector into Wayne State University Libraries' catalog that displayed library books as they appeared on the physical bookshelf. This backend connector worked over the Z39.50 protocol, transforming MARC records into json data. Also, each record, when displayed, queries against the Google Books and Open Library APIs to find available physical or digital copies.

  • Client: Wayne State University Libraries
  • Date: July 2013
  • Service: Web Development, PHP, Z39.50, MARC, JSON, RESTFul APIs

Ref Stats

Along with my colleague, Graham Hukill, I built a statistics tracker for the Wayne State University Libraries' reference department. Built upon PHP, jQuery, bootstrap, and a MySQL backend, it allows for quick entering, editing, and vizualizing of activity at campus reference desks.

  • Client: Wayne State University Libraries
  • Date: October 2014
  • Service: Web Development, PHP, user testing, MySQL, bootstrap, CRUD interfaces

Ouroboros

A management tool built to handle the bulk ingest and display of digital objects in a Fedora Commons repository. Named Ouroboros (a nod to its Python codebase), the system allows for object ingest, metadata management, and object modeling. Ouroboros is built in Python using the Flask framework and Twisted web server.

  • Client: Wayne State University Libraries
  • Date: October 2013 - Present
  • Service: Web Development, Python, Flask, Redis, Twisted, eulfedora, Fedora Commons, Solr

WSU Digital Collections

Built by the WSU Libraries Digital Publishing unit, the WSU Digital Digital Collections site is a lightweight PHP, HTML, CSS front-end to display the digital colletions of Wayne State University Libraries. The front-end queries a Fedora Commons/Solr backend through the use of a custom RESTful API. The website can be found at http://digital.library.wayne.edu.

  • Client: Wayne State University Libraries
  • Date: November 2012 - Present
  • Service: Web Development, PHP, jQuery, Mustache, MySQL, bootstrap
